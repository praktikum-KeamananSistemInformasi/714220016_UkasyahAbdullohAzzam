How AES Works:

AES (Advanced Encryption Standard) uses a block cipher technique with a fixed block size of 128 bits and key sizes of 128, 192, or 256 bits. AES operates in different modes, and we are using the EAX mode, which provides both confidentiality and authenticity.

{% elif algorithm == 'DES' %}

How DES Works:

DES (Data Encryption Standard) is a block cipher that encrypts data in 64-bit blocks using a 56-bit key. It is considered less secure than modern standards like AES due to its shorter key length.

{% elif algorithm == '3DES' %}

How 3DES Works:

3DES (Triple DES) is a more secure version of DES, which applies the DES algorithm three times to each data block. This makes it significantly more secure than standard DES.

{% elif algorithm == 'Blowfish' %}

How Blowfish Works:

Blowfish is a symmetric-key block cipher that uses variable-length keys (32 to 448 bits) and encrypts data in blocks of 64 bits. It is known for being fast and secure.

{% elif algorithm == 'ChaCha20' %}

How ChaCha20 Works:

ChaCha20 is a stream cipher known for its efficiency and security. It operates by generating a keystream that is XORed with the plaintext to produce the ciphertext.

{% elif algorithm == 'Salsa20' %}

How Salsa20 Works:

Similar to ChaCha20, Salsa20 is a stream cipher that is fast and secure. It generates a keystream for encryption by XORing it with the plaintext.

Why You Can't Use Different Algorithms for Encryption and Decryption:

Each encryption algorithm operates in a specific way, generating encrypted text based on its unique structure, key size, and block size (for block ciphers). When data is encrypted using one algorithm, such as AES, the output (ciphertext) is dependent on the specific operations that AES performs.

If you try to decrypt the ciphertext using a different algorithm (e.g., trying to decrypt AES-encrypted data with Blowfish), the structure of the ciphertext won't match the expected format of the decryption algorithm. This mismatch results in an error because the decryption algorithm cannot correctly interpret or reverse the operations of the original encryption algorithm.
